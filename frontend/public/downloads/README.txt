Place the built Android APK here as: nexusllm.apk

The website's "Download App" button links to /downloads/nexusllm.apk
(i.e. this folder). After you build the APK from the LLMFree-App project
(Android Studio ▸ Build ▸ Build APK(s)), copy/rename it to:

    frontend/public/downloads/nexusllm.apk

then commit + push so it's downloadable from the deployed site.
